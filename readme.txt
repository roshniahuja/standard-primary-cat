== Description ==

Set a primary category for your (custom) posts and query them in your template using shortcode [posts_by_primary_category category="primary-category"].

== Installation ==

1. Upload the `standard-primary-cat` folder to your `/wp-content/plugins/` directory
2. Activate the plugin via the Plugins menu in WordPress
